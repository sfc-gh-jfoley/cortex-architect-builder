This plugin provides a multi-agent project framework for building any software project with coordinated agents.

- **agent-architect** — Orchestrates Architect/Security/Researcher/Worker/Tester roles via manifest.json state bus, git+GitHub integration, and gated handoffs

To enable: `cortex plugin enable cortex-architect-builder`

Start with: `$cortex-architect-builder:agent-architect`

Trigger phrases: architect project, build project, assign teams, I want to build, agent architect, offshore team
